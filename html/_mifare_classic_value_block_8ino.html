◆ RST_PIN

#define RST_PIN   9

This is a MFRC522 library example; see https://github.com/miguelbalboa/rfid for further details and other examples.

NOTE: The library file MFRC522.h has a lot of useful info. Please read it.

Released into the public domain.

This sample shows how to setup blocks on a MIFARE Classic PICC (= card/tag) to be in "Value Block" mode: in this mode the operations Increment/Decrement, Restore and Transfer can be used.

BEWARE: Data will be written to the PICC, in sector #1 (blocks #4 to #7).

Typical pin layout used:
