connections

SDA | RX | SS is connected to SS_pin SCK is pin 52 (Mega2560) MOSI is pin 51 (Mega2560) MISO is pin 50 (Mega2560) RST | Reset is connected to RST_pin LED is connected to LED_pin and is active high The LED should have a resistor to limit current (Using 220R, variation changes light level)
